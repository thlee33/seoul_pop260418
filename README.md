# 서울시 생활인구 3D 시각화 프로젝트 (250m 격자)
**Seoul Living Population 3D Visualization on 250m Grid**

![MapLibre](https://img.shields.io/badge/MapLibre-GL_JS-orange?style=for-the-badge)
![React](https://img.shields.io/badge/React-18-cyan?style=for-the-badge)

본 프로젝트는 **서울시 열린데이터광장**에서 제공하는 **250m 격자 단위의 생활인구 데이터**를 활용하여, 주말(토요일) 시간대별 내국인과 단기체류 외국인의 유동인구 패턴을 3D 지도로 직관적으로 비교·분석할 수 있도록 개발된 정적 웹 애플리케이션입니다.

## 📊 데이터 소스 및 처리
- **데이터 출처**: 서울시 생활인구 (250m 격자) - 2026년 4월 18일(토요일) 데이터 기준.
- **생활인구란?**: 거주지 중심의 통계가 아닌, 특정 시점에 해당 지역에 머물고 있는 인구를 **통신 데이터 기반으로 추정**한 실질적인 유동/체류 인구입니다.
- **데이터 처리**: 
  - `EPSG:5179` 좌표계로 제공되는 250m 격자 폴리곤 중심점을 `WGS84(EPSG:4326)` 경위도 좌표계로 변환.
  - 내국인 및 외국인 생활인구 CSV 데이터를 공간 데이터와 병합하여 최적화된 JSON 구조로 추출.

## 🚀 주요 기능 (Features)
1. **GPU 기반 3D 시각화 (deck.gl)**
   - 수천 개의 250m 격자를 브라우저에서 버벅임 없이 GPU로 렌더링.
   - `Shift + 마우스 드래그`를 통한 3차원 지도 회전 및 기울기(Pitch) 조절.
2. **내국인 vs 외국인 듀얼 렌더링 & 동적 단계구분도**
   - **겹침 방지 설계**: 동일한 격자 내에서도 내국인은 서쪽, 외국인은 동쪽으로 이격된 직사각형 기둥으로 표현하여 두 집단의 분포를 한눈에 비교.
   - **Choropleth Map**: 인구 밀집도에 따라 하위 20%부터 상위 20%까지 분위수(Quantiles)를 동적으로 계산하여 5단계 색상(내국인: Blues, 외국인: Oranges) 적용.
3. **타임 슬라이더 & 자동 재생**
   - 하단 슬라이더를 통해 00시부터 23시까지 시간대별 인구 이동 패턴을 조회.
4. **실시간 랭킹 라벨링 (Top 10)**
   - 선택된 시간대별로 인구가 가장 많은 최상위 10개 격자를 자동으로 계산하여 기둥 위에 순위(1~10) 라벨 표시.

## 💡 주요 인사이트 (Key Insights)
데이터 시각화 및 분석을 통해 다음과 같은 유의미한 주말(토요일) 생활인구 패턴을 도출할 수 있었습니다.

1. **내국인: 대형 상업/여가 시설 중심의 주간 밀집**
   - **최대 밀집 시간**: 오후 3시 (15:00) 경 서울 전체 생활인구 활동량이 최고조(약 1,169만 명)에 달함.
   - **최대 밀집 지역**: 단일 250m 격자 기준, **잠실 롯데월드타워 및 롯데월드몰** 일대에 약 3만 명 이상이 집중되는 현상 확인. 이는 주말 오후 거대한 집객 시설이 도심 유동인구 블랙홀로 작용함을 보여줌.
2. **단기체류 외국인: 야간 관광 및 핵심 상권 밀집**
   - **최대 밀집 시간 및 지역**: 전체 외국인 생활인구 총합은 오후 2시에 가장 많았으나, 특정 단일 격자로의 극단적 밀집은 오후 8시(20:00)에 나타남 (단일 격자 최대 약 3,400명).
   - 이는 주간에는 고궁, 랜드마크 등 서울 전역으로 넓게 퍼져 관광을 즐기다가, 저녁 식사 및 야간 여가/쇼핑을 위해 명동, 홍대, 이태원 등 **특정 핵심 상권으로 강하게 응집**하는 관광객 특유의 패턴을 방증함.
3. **공간적 분리 현상**
   - 내국인은 주거지와 상업지 전반에 걸쳐 넓은 분포를 보이지만, 외국인은 특정 핫플레이스와 호텔 밀집 지역을 따라 극도로 편향된 클러스터를 형성하고 있음.
